# AMIRA ALDHAD - Luxury Gold & Investment Website

A modern, responsive website for AMIRA ALDHAD luxury gold jewelry and investment opportunities.

## Features

- ✨ Modern, elegant design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Fast performance with Vite
- 🎨 Beautiful UI with Tailwind CSS
- 🔄 Smooth animations and transitions
- 📧 Contact form integration
- 🌐 Multi-channel communication

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Lucide React Icons

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone repository
git clone https://github.com/Airo782/amira-aldhad-gold.git
cd amira-aldhad-gold

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
